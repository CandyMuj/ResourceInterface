# 备用地址

> 由于主地址接了Cloudflare，部分时候或地区会存在 **无法访问或加载缓慢的问题，建议使用备用地址**
>
> **备用地址：** 就是GitHub的地址，加了个加速服务，如果备用地址也用不了了，可以自己百度一下， **找一个能用的加速服务进行替换即可**
